## [Storybook](https://storybook.js.org) test.

|`yarn run <script>`|What it does|
|------------------|------------|
|`start`|Starts the app at [http://localhost:3000/](http://localhost:3000/). The bundle js/css files are stored in memory|
|`storybook`|Starts the storybook at [http://localhost:9009/](http://localhost:9009/). The bundle js/css files are stored in memory|
